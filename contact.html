Important Information

Scam Warning

We do not sell access via Discord, Telegram, or any third-party platform. Always verify staff members through official forum PMs. If someone contacts you claiming to be staff outside of the forum, it is likely a scam.

Response Times

Support requests are typically answered within 24-48 hours. VIP members receive priority support.

Before Contacting Us

  • Check the Guides for setup, calibration, and troubleshooting steps
  • Run pixel --validate-profile to check your profile for config errors
  • Review your telemetry logs in the logs/ directory for error details
  • Search the forums for similar issues
  • Make sure you're using the latest Pixel build
  • Include relevant details (error messages, telemetry excerpts, screenshots) in your request
